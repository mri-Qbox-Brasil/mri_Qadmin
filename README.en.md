# MRI QAdmin

Professional and modern Admin Panel for Qbox/QbCore servers.

## Features

- **Dashboard**: Overview of server status and metrics.
- **Player Management**:
  - Complete list of online/offline players.
  - Quick actions: Revive, Heal, Kill, Freeze, Spectate, Teleport, etc.
  - Moderation: Ban, Kick, Warn.
  - Economy: Give/Remove Cash, Bank, Crypto.
  - Inventory: View, Clear, Give Items.
  - Vehicles: View player vehicles, Spawn, Delete, Open Trunk.
- **Group Management**: Manage Jobs and Gangs.
- **Bans System**: Comprehensive ban list and management.
- **Vehicle Management**: Admin vehicle spawner and management.
- **Items Database**: Search and give items.
- **Server Resources**: Monitor and manage server resources.
- **Staff Chat**: Integrated chat for staff members.
- **Developer Tools**:
    - Entity info, coordinates, and more.
    - **Dynamic Wall (ESP)**: Custom colors for Dead players, Invisible players, and by permission groups (ACE Principals).
- **Customizable**:
    - Dark/Light modes.
    - Custom accent colors (Hex, RGB, HSL support).
