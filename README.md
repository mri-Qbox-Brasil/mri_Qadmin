# MRI QAdmin

Painel Administrativo profissional e moderno para servidores Qbox/QbCore.

[Read in English](README.en.md)

## Funcionalidades

- **Dashboard**: Visão geral do status do servidor e métricas.
- **Gerenciamento de Jogadores**:
  - Lista completa de jogadores online/offline.
  - Ações rápidas: Reviver, Curar, Matar, Congelar, Espectar, Teleportar, etc.
  - Moderação: Banir, Kickar, Avisar (Warn).
  - Economia: Adicionar/Remover Dinheiro, Banco, Crypto.
  - Inventário: Visualizar, Limpar, Dar Itens.
  - Veículos: Ver veículos do jogador, Spawnar, Deletar, Abrir Porta-malas.
- **Gerenciamento de Grupos**: Gerenciar Empregos (Jobs) e Gangues.
- **Sistema de Banimentos**: Lista completa e gerenciamento de banimentos.
- **Gerenciamento de Veículos**: Spawnador de veículos administrativos e gerenciamento.
- **Banco de Dados de Itens**: Pesquisar e dar itens.
- **Resources do Servidor**: Monitorar e gerenciar os resources do servidor.
- **Chat da Staff**: Chat integrado para membros da equipe.
- **Ferramentas de Desenvolvedor**: Informações de entidades, coordenadas e mais.
- **Personalizável**:
    - Modos Claro/Escuro.
    - Cores de destaque personalizadas (Suporte a Hex, RGB, HSL).
